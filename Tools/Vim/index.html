# Vim 总结

  1. 安装
  2. 下载
  3. 安装
  4. 配置
  5. 主题
  6. 字体
  7. 插件管理 vundle
  8. 概念
  9. 模式：标准模式 normal、插入模式 insert、视图模式 visual、命令模式 command（命令行 command line、命令窗口、操作符待决 operator-pending）
  10. 命令语法：operator + motion （动词 + 介词/量词 + 名词）
  11. 操作范围：文本对象（字符、单词、字串）
  12. 缓冲区
  13. 寄存器：无名寄存器、复制专用寄存器、有名寄存器、黑洞寄存器、系统剪贴板
  14. 实际行和屏幕行
  15. 虚拟替换
  16. 前缀 leader
  17. 基本操作
  18. 启动
  19. 退出
  20. 打开文件
  21. 保存文件 :w
  22. 另存为文件
  23. 标签页操作
  24. 窗口操作 :split :vsplit :close alt+hjkl
  25. 标准模式下操作：先指定操作，再确定范围
  26. 操作光标移动：字符 h j k l 单词 end word begin 字串 W B 行首0 ^(H) 行尾 $(L) 窗口顶部 H(^) 窗口底部 L($) 窗口中间 M 文档首行 gg 文档尾行 G $G 隔行 :n 或 nG 或 ngg 多列 n| 括号切换 %
  27. 滚动屏幕: 向上一页 C+f 向下一页 C+b 向上半页 C+u 向下半页 C+d 向上一行 C+e 向下一行 C+y 使当前行滚动到窗口顶部 zt 使当前行滚动到屏幕中间 zz 使当前行滚动到屏幕底部 zb
  28. 查找
  29. 字符 find：向下查找 f 向上查找 F 向下查找目标字符之前 t 同上查找目标字符之后 T ; 跳转下一个目标字符 , 跳转上一个目标字符
  30. 单词 * 向下查找光标所在单词 # 向上查找光标所在单词
  31. 搜索字符串，包括单词 / 向下搜索 ？向下搜索 n 跳转下一个搜索到的目标 N跳转到上一个搜索的目标 回到原来光标位置（待） 退出当前搜索高亮（:noh）
  32. 替换 substitute：列范围s替换/目标/行范围 列范围中, 和 ; 区别 c手动确认
  33. 删除 字符向后一个 x 字符向前一个 X 单词 dw 行 dd 连续多行 nd
  34. 复制 yank 复制当前行 yy
  35. 粘贴 paste 在后粘贴 p 在前粘贴 P
  36. 撤消 u 重做 U
  37. 重复上次操作 .
  38. normal 命令 [rang]normal motion
  39. 视图模式下操作: 先选择范围，再执行操作
  40. v visual mode
  41. V visual line mode
  42. ctrl + v visual block mode
  43. 切换块的起点和终点 o
  44. 交换块区域的起点和终点 O
  45. 选择当前单词 aw
  46. 选择当前标签包围区域 at it
  47. 选择当前 （）区域 ab ib
  48. 选择当前 {} 区域 aB iB
  49. 回到上次块区域 gv
  50. 插入模式下操作
  51. 字符前插入 i 字符后追加插入 a 行首插入 I 行尾追加插入 A 向上插入空行 O 向下插入空行 o
  52. 删除当前字符并插入 s 删除当前行并插入 S
  53. 修改c+motion 并插入 C 删除光标至行尾并插入
  54. 替换插入 replace : r 命令：替换当前字符 R 命令：切换到替换模式，同键盘上的inster键一样功能
  55. 插入模式下移动光标：alt + hjkl home end
  56. 命令行模式下操作 home end 上下左右箭头
  57. 命令窗口下操作（同插入模式一样移动）

# vimium 操作

  1. 标签页 前一个 k 后一个 J 上一个 E 第一个 S 最后一个 D 关闭 x 恢复上一个关闭 X 新建一个 t 切换 T 弹出标签为独立窗口 M 向右移动 >> 向左移动<<
  2. 地址栏（历史记录、刷新、URL） 前一个历史 H 后一个历史 L 刷新 r 编辑当前URL ge gE 复制当前URL yy 粘贴在当前地址栏 p 新开标签URL P
  3. 书签，搜索 b B o O
  4. 窗口 单行移动 j k 半页滚动 s d 顶部 gg 底部 G 聚集第一个输入框 gi
